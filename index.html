Private Email Server

This is a self-hosted server for a private email service. You can create custom email addresses to send and receive emails from.

Getting Started

  1. Install the software on your server.
  2. Create a new email account.
  3. Configure your email client to use the new email account.

Features

  • Custom email addresses
  • Send and receive emails
  • Webmail interface
  • IMAP and POP3 support
  • SMTP support
  • Encryption

Security

The server is protected by a firewall and uses encryption to protect your data.
